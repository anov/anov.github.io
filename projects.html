Predicting Public Opinion in 140 Characters or Less

I'm working on an interdisciplinary project in Political Science and Computer Science, seeking to address whether Twitter can tell us something about public opinion, and how public opinion expressed on Twitter compare to traditional public polling? Code for the project lives on Github. I began the project working in Python and planning on using nltk and R for the machine learning and stat-crunching aspects of the project, respectively. I ended up migrating to an entirely Python-based stack including pandas, scikit-learn, and statsmodels, with a workflow that made increasing use of IPython.
